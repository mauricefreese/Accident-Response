Accident Response

This product is designed for the safety of drivers worldwide. This sample device is made from a raspberry pi. It is designed to be put in cars and save lives. This device tracks location with a onboard gps and it also can can notify emergency services when there is a high impact collision and/or fire.

Why this?

This is not only a safety device, but it is a device that can be used for monitoring. It can be used to notify emegency sevices when there is a fire or accident. Also, it can notify parents or employers if their vehicle leaves a certain display area.
